# pureadapter
Pure RecyclerView Adapter
万能的RecyclerView Adapter

# 如何使用

## 1.在根目录的build.gradle中添加如下代码：
```java
allprojects {
    repositories {
        ...
        maven { url 'https://jitpack.io' }
    }
}
```

## 2.添加依赖
```java
dependencies {
    compile 'com.github.laucherish:pureadapter:1.2.0'
}
```
